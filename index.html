Formatação de Textos em CSS

font-family: --> Família de fontes
font-size: --> Tamanho da fonte
text-decoration: --> Decoração de textos
line-height: --> Propriedade especifica a altura de uma linha.
color: --> Cor do texto
width: --> Largura
height: --> Altura
border: --> Bordas
background-color: --> Cor de Fundo
margin: --> É a margem do elemento, ou seja, o espaçamento externo do elemento.
font-weight: --> A propriedade define como os caracteres grossos ou finos no texto devem ser exibidos.
border-radius: --> Define o raio dos cantos do elemento.
